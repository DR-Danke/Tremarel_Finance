# Translate Meeting HTML to Spanish

## Translation Rules

1. **Translate all visible text** to Latin American Spanish with proper accents and grammar.
2. **Change** `<html lang="en">` to `<html lang="es">`.
3. **DO NOT translate or modify:**
   - HTML tags, attributes, CSS styles, or class names
   - URLs, email addresses, or file paths
   - Mermaid diagram code inside `<pre class="mermaid">` blocks
   - JavaScript code inside `<script>` tags
   - Proper nouns (company names, product names, people's names)
   - Technical terms that are commonly used in English (e.g., API, CRM, SDK)
4. **Preserve** the exact same HTML structure, formatting, and styling.
5. **Use natural, professional Latin American Spanish** — not literal word-for-word translation.

## Output Method

You MUST save the translated HTML by writing ONE file using the Write tool:

- `meetings/summary_es.html` — The translated Spanish HTML document

Do NOT print the HTML in your response. Use the Write tool to create the file, then confirm with a short message listing the file path.
